--- layout: default title: Visage ---

Visage is a web interface for viewing collectd statistics.

It also provides a JSON interface onto collectd's RRDs, giving you an easy way to mash up the data.

Download

Code and install instructions live on GitHub.

Graphs

You should see a graph rendered below. Try moving your mouse over one of the labels.

You can select a timeframe to view with the dropdown in the top right. Try clicking within the graph to select a timeframe.
